# moodgraph.io

[moodgraph.io](https://moodgraph.io/) is a progressive web app built with [Next.js](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/).

## Installation

```bash
git clone https://github.com/moodgraphio/moodgraph.io

yarn install
yarn dev
```
